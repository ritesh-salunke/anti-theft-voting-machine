# Anti-Theft Voting Machine

## 📖 Overview
This project is an embedded-system-based secure voting machine designed using ATmega328P microcontroller and fingerprint authentication for secure voting operations.

---

## 🚀 Features
- Fingerprint-based voter authentication
- Secure voting process
- Anti-theft protection mechanism
- EEPROM-based data storage
- User-friendly operation

---

## 🛠 Components Used
- ATmega328P
- R307 Fingerprint Sensor
- LCD Display
- Push Buttons
- Buzzer
- Power Supply Module

---

## 💻 Software & Tools
- Arduino IDE
- Proteus
- Embedded C

---

## ⚙️ Working
1. User authentication using fingerprint module
2. Valid voter verification
3. Vote registration through push buttons
4. Secure vote storage in EEPROM
5. Anti-theft protection activates during unauthorized access
